The simple rule
At each iteration, the app places smaller pyramid-shaped bumps across the surface. In the perfect case, every bump is centered and every height follows the same decay rule. Randomness changes where bumps land, how tall they are, whether some are skipped, and whether some become valleys instead of peaks.
